# Ticket Triage Mini Project

This Python mini-project simulates a ticket triage system using a mock AI environment. The goal is to automatically classify and prioritize support tickets based on their content.  

It is ideal for demonstrating basic AI/ML workflow concepts in a safe, demo-friendly environment without relying on external APIs.

## Features

- Accepts dynamic user input for ticket descriptions.
- Uses keyword-based mock logic to classify tickets:
  - High priority: urgent, failure, critical
  - Medium priority: slow, issue, problem
  - low priority: general requests or minor messages
- Outputs:
  - Suggested action
  - Category (critical, performance, general)
  - Priority (high, medium, low)
  - Score (mock confidence value)
